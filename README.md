# UniMarket
An efficient University Marketplace built with Django and Bootstrap, designed to help students buy and sell used textbooks and university equipment.

## ✨ Features
- Product Management: Full CRUD operations through Django Admin.
- Dynamic Frontend: Displays available products in real-time.
- Responsive Design: Clean UI using Bootstrap 5.
- Backend Architecture: Built using the MTV (Model-Template-View) pattern.

## 🛠️ Tech Stack
- Backend: Python, Django 5.x
- Frontend: HTML5, CSS3, Bootstrap 5
- Database: SQLite (Development)

## 🚀 How to Run Locally
1. Clone the repository: git clone [https://github.com/fai-def/UniMarket.Django.git]
2. Create a virtual environment: python -m venv venv
3. Activate venv: .\venv\Scripts\activate
4. Install Django: pip install django
5. Run migrations: python manage.py migrate
6. Start the server: python manage.py runserver
