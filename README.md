# UniMarket - University Marketplace

A lightweight, efficient e-commerce platform built with Django for university students to exchange textbooks, tools, and equipment.

## ✨ Features
- Dynamic Search Engine: Search for products by name or description using Django's Q-objects.
- User-Friendly Interface: A clean, responsive design using Bootstrap 5.
- Admin Dashboard: Full control over product listings, prices, and categories.
- RTL Support: Optimized for Arabic language display while maintaining a professional look.

## 🛠️ Tech Stack
- Backend: Python / Django
- Frontend: HTML5, CSS3, Bootstrap 5
- Database: SQLite3
- Version Control: Git & GitHub

## 🚀 How to Run Locally
1. Clone the repository: git clone https://github.com/fai-def/UniMarket.Django.git
2. Create virtual environment: python -m venv venv
3. Activate environment: .\venv\Scripts\activate
4. Install Django: pip install django
5. Run migrations: python manage.py migrate
6. Start the server: python manage.py runserver
